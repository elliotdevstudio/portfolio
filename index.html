PROJECTS



GRAT - restaurant tip distribution system

Full-stack SaaS application built with Next.js 15 (App Router), TypeScript, and MongoDB Atlas that automates complex tip pooling calculations for restaurants with multi-tier gratuity distribution. Architected bidirectional data relationships between staff groups using atomic state management (Jotai), implemented server-side calculation logic for percentage-based and pooled tip distribution across multiple recipient tiers, and designed a 90-day rolling data retention system with automated CI/CD maintenance via GitHub Actions. Features include aggregated shift analytics with flexible date range filtering, CSV export functionality, and a responsive UI built with Tailwind CSS. Deployed on Vercel with production-ready error handling, environment-based configuration, and daily automated shift generation.

Technologies used:  Next.js(App router), Tailwind, Typescript, Jotai state management, Github Actions, MongoDB Atlas

FIG TREE - a hackathon project

This collaborative hackathon was hosted by General Assembly in May 2023, with a theme of 'Digital Equity and Accessiblity'. Our team of two developers and two UX/UI designers determined that the ubiquity and affordability of mobile smart devices opened some ground for exploring ideas, and with guidance of publicly available peer-reviewed research we concluded that our product should aim to assist individuals facing significant economic or housing challenges. To effectively manage the extensive range of desired components and functionalities, we focused on developing a minimum viable product (MVP) that included user authorization, a basic profile screen allowing users to create personal profiles, and a landing page featuring publicly accessible routes to various resources generated during our brainstorming sessions. For a few other additional endpoints that fell outside the scope of our project, we created mock-ups for presentation purposes, incorporating user interactions such as drop-down menus and basic animations. Our team was one of the smallest groups, after half of the initial participants needed to disengage. This was a great experience that challenged us on a professional level.

Technologies used:  Express.js, React.js, MongoDB, Netlify, CSS modules, HTML





FEEDBACK - A replica of Facebook's 'comment' feature

This application runs on a Node server using REST architecture, and leverages more advanced tools and functionality of the Express.js framework, such as user authorization through session logging and bcrypt for password hashing. It uses middleware protected routes, connects to an external MongoDB atlas database, Mongoose for an ODM, is rendered with EJS (extended javascript) templates, and styled using Bulma and custom CSS. This app essentially allows users to post text blocks on a chronological feed that anyone who is logged in can post comments to, and is CRUD functional.

Technologies used:  Node.js, Express, MongoDB, SASS, HTML5
